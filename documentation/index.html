Getting Started

Unzip files from archive and you will find Html folder. Inside you'll find following files and folders:

  • /css - folder with css files
  • /fonts - folder with icon fonts
  • /fonts/flaticont - folder with icon fonts
  • /img - folder with all images
  • /js - folder with jQuery script files

Note: Contact form will not work offline. You must upload your website on a server

Contact Form

Open contact.html with any text editor and change # then paste your email address like 

 <form class="form" action="themestok.com@gmail.com" method="POST">// Your email past here
in line 178. Save it.

Note: Contact form will not work offline. You must upload your website on a server

Changing Logo

Changing Colors

There are 2 ways to change main color of theme

Option 1

Manually change all color values in style.css Open text editor and change the colors value.

Option 2

Manually change all color values in assets/sass/base/_variable.scss Open text editor and change the colors value.

Changing Icons

To change icons follow these steps.

Step 1.

Choose desired icon in the list - link

Step 2.

Get the class name of the icon

Step 3.

Paste the class in icon
